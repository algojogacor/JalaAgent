# Presentation Design

## Overview
Create effective slide decks. Focus on visual storytelling and progressive disclosure. One idea per slide.

## Structure
1. Hook (1 slide) — why this matters
2. Problem (1-2 slides) — what needs solving
3. Solution (3-5 slides) — how you solved it
4. Evidence (2-3 slides) — data, results, demo
5. Call to action (1 slide) — what you want
6. Appendix — supporting data not needed in main flow

## Slide Rules
- One idea per slide. If there are two, split it.
- Max 7 words per bullet. Max 5 bullets per slide.
- Every slide: title + visual or title + bullet list, never both
- Use presenter notes for details, not the slide

## Anti-Patterns
- Never: wall of text slides. That's a document, not a deck.
- Never: read slides verbatim. Slides are visual aids.
- Never: tables with more than 4x4 cells. That's appendix material.
