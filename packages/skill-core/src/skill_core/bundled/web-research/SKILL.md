# Web Research

## Overview
Conduct multi-source research and synthesize findings. Evaluate source credibility. Cross-reference claims.

## Process
1. Define research question (specific, not vague)
2. Identify 3+ independent sources per claim
3. Evaluate each source: authority, currency, bias, evidence
4. Cross-reference: do sources agree? Where do they differ?
5. Synthesize: what's the consensus? What's contested?
6. Cite: every factual claim needs a source link

## Source Evaluation
- Authority: who wrote it? Credentials?
- Currency: when was it published/updated?
- Bias: what's their incentive?
- Evidence: do they cite primary sources?

## Anti-Patterns
- Never trust a single source for a factual claim
- Never cite Wikipedia directly (use its references)
- Never present speculation as fact
