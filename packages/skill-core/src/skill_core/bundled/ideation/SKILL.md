# Ideation

## Overview
Generate project ideas using structured creativity techniques. Constraints produce better ideas than open-ended brainstorming.

## Techniques
1. **SCAMPER** — Substitute, Combine, Adapt, Modify, Put to another use, Eliminate, Reverse
2. **Combinatorial** — Take two unrelated products/concepts and merge them
3. **Constraint-first** — Define limits (budget, time, platform, audience) then ideate within them
4. **Inversion** — What would the worst version look like? Invert it.

## Process
1. Define the domain/problem space (1 sentence)
2. List 5 constraints (technical, time, audience, budget, platform)
3. Generate 10 ideas within those constraints
4. Score each: novelty (1-5), feasibility (1-5), impact (1-5)
5. Present top 3 with 1-paragraph pitch each

## Anti-Patterns
- Don't reject ideas during generation (separate generation from evaluation)
- Don't ideate without constraints (produces obvious ideas)
- Don't present only one idea (always give options)
