# Creative Writing

## Overview
Write compelling fiction and narrative content. Focus on concrete techniques: showing vs telling, pacing, dialog, sensory detail.

## Process
1. Define: genre, tone, target length, POV (1st/3rd/omniscient)
2. Outline: 3-act or 5-act structure with key beats
3. Character: motivation, flaw, arc (how they change)
4. Draft: write first, edit later
5. Edit: cut 10% minimum, check passive voice, vary sentence length

## Techniques
- Show, don't tell: "Her hands shook" not "She was nervous"
- Dialog: each character has distinct vocabulary and rhythm
- Sensory: every scene includes at least 2 senses
- Pacing: short sentences for action, long for reflection

## Anti-Patterns
- Don't start with backstory (start in the middle)
- Don't use adverbs as crutches ("said loudly" → "shouted")
- Don't head-hop within a scene (stick to one POV)
- Don't explain the moral (let the reader infer)
