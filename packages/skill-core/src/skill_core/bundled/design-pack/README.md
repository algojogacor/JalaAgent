# Design Skills Pack

> Bundled from Hermes creative/ skills. Easy to remove: rm -rf bundled/design-pack/

## Skills included

| Skill | Description |
|-------|-------------|
| animated-gradient-mesh | CSS animated gradient mesh backgrounds |
| architecture-diagram | System architecture diagrams via Excalidraw/Mermaid |
| design-taste | Design critique and taste development |
| micro-interactions | CSS/JS micro-interactions for UI polish |
| popular-web-designs | Analyze popular web designs, extract patterns |
| open-design | Open-source design system tokens and components |
| design-md | Markdown-based design documentation |
| excalidraw | Excalidraw diagram creation and editing |
| sketch | SVG sketch/illustration generation |
| p5js | Creative coding with p5.js |

## Removal

To remove all design skills: rm -rf packages/skill-core/src/skill_core/bundled/design-pack/
