# Popular Web Designs

Extract and apply design patterns from successful websites.

## Analysis Process
1. Identify the site's primary goal (conversion, engagement, information)
2. Map the visual hierarchy (what draws the eye first?)
3. Extract: typography pairing, color palette, spacing system
4. Identify reusable patterns: navigation, cards, CTAs, forms
5. Adapt, don't copy — extract the principle, not the pixels

## Current Trends (2026)
- Bento grid layouts (Apple-style modular grids)
- Dark mode as default, light as toggle
- Variable fonts for responsive typography
- Glassmorphism 2.0 (subtle blur, not heavy)
- Micro-interactions on scroll (intersection observer)

## Anti-Patterns
- Never copy a design wholesale (adapt the pattern)
- Never apply multiple trends to one page
- Never skip usability for aesthetics
