# Design Taste

Good design follows learnable patterns, not subjective whim.

## Critique Framework
For any design, ask:
1. **Hierarchy**: What do you see first? Second? Is that correct?
2. **Spacing**: Is whitespace consistent? Related items grouped?
3. **Typography**: Clear type scale? Max 2 fonts?
4. **Color**: One accent color? Neutrals consistent?
5. **Intentionality**: Every element has a reason to exist?

## Quick 10-Second Fixes
- Line-height 1.5 for body text
- Add 8px more padding to cards
- Remove one color from the palette
- Make CTA 20% larger
- Add subtle border between sections

## Anti-Patterns
- Equal spacing everywhere (use proximity to group)
- Center-aligning body text (left-align for readability)
- Pure black on pure white (use near-black/near-white)
